# 우리반 마피아 웹 v2

학교 PC와 학생 태블릿이 같은 와이파이가 아니어도 사용할 수 있는 웹용 버전입니다.

## GitHub 업로드
1. GitHub 저장소에서 Add file > Upload files를 누릅니다.
2. index.html, app.js, style.css 파일을 업로드합니다.
3. Commit changes를 누릅니다.

## GitHub Pages 켜기
1. 저장소 상단 Settings
2. 왼쪽 Pages
3. Deploy from a branch 선택
4. Branch: main, Folder: /root 선택
5. Save
6. 잠시 기다린 뒤 표시되는 주소로 접속합니다.

## 사용
1. 웹 주소로 접속
2. 선생님: 새 방 만들기
3. 학생: 선생님 화면에 나오는 학생 접속 주소로 입장
4. 이름 입력 후 대기
5. 선생님이 역할 배정 후 밤/낮/투표 진행

## 역할 비율
20~22명: 마피아 4, 경찰 1, 의사 1, 나머지 시민
19명 이하: 마피아 3, 경찰 1, 의사 1, 나머지 시민

## 특징
- 역할 3초 보기 버튼
- 마피아 비밀 대화
- 밤 행동, 투표, 처형
- 관리자 수동 처형/되살리기
- 자동 승리 판정

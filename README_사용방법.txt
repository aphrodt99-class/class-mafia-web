# 우리반 마피아 웹 v9 - 실제 플레이 피드백 반영

## v9 수정 내용
- 채팅 입력 중 다른 학생 메시지 때문에 입력창이 사라지는 문제 완화
- 채팅창은 새 글이 아래로 쌓이고 자동으로 아래쪽을 보여주도록 조정
- 투표는 1회만 가능, 변경 불가
- 모두 투표하면 학생/관리자 화면에 “모두 투표했습니다” 표시
- 밤에 사망하거나 투표로 처형되면 역할 공개
- 경찰/의사/마피아에게 밤에만 본인 전용 비밀 알림 표시
- 마피아 밤 선택이 동점이면 첫 번째로 선택한 마피아의 선택을 기준으로 처리
- 세라반 고정방 SERA57 유지
- 학생 QR 고정 유지

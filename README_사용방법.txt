# 우리반 마피아 웹 v6 - 세라반 전용 고정방

## v6 핵심
- 방 코드 고정: SERA57
- 학생 QR 고정: 새 방을 만들 때마다 QR을 다시 만들 필요 없음
- 관리자 화면에 학생 접속 QR 자동 표시
- 새 판 시작(학생 유지): 학생 명단은 그대로 두고 역할/채팅/투표/결과만 초기화
- 전체 초기화(학생 삭제): 학생 명단까지 모두 삭제
- 20~24명 기준: 마피아 4명, 경찰 1명, 의사 1명, 나머지 시민
- 19명 이하 테스트/소규모 기준: 마피아 3명, 경찰 1명, 의사 1명, 나머지 시민

README_사용방법.txt는 선택입니다.

## 사용 흐름
1. 웹 주소 접속
2. 선생님: 세라반 관리자 입장
3. 학생: 학생으로 입장 또는 관리자 화면 QR 스캔
4. 학생 이름 입력
5. 역할 배정 & 1일차 밤 시작
6. 게임 종료 후 다음 판은 관리자 화면의 새 판 시작(학생 유지) 사용
7. 명단까지 지우고 싶으면 전체 초기화(학생 삭제) 사용

# 우리반 마피아 웹 v10 - 사망자 대화방 추가

## v10 수정 내용
- 죽은 학생 전용 채팅방 추가
- 죽은 시민 편과 죽은 마피아 채팅방 분리
- 살아있을 때는 보이지 않고, 죽었을 때만 활성화
- 죽은 사람은 살아있는 사람에게 힌트를 주지 말고 사망자 채팅방에서 대화하도록 안내
- 기존 기능 유지:
  - 세라반 고정방 SERA57
  - 학생 QR 고정
  - 투표 1회 제한
  - 투표 로그 공개
  - 처형/사망 시 역할 공개
  - 마피아 동점 시 첫 선택 기준 처리
